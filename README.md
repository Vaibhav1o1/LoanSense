# LoanSense-
LoanSense — Intelligent Loan Approval Predictor
A Machine Learning project that predicts whether a loan application should be approved or rejected based on applicant data.
📊 Model Performance
MetricScoreAccuracy82%Precision80.35%Best AlgorithmNaive Bayes
🎯 Problem Statement
Loan approval is a high-stakes financial decision. A wrong approval can cost lenders significantly. This project builds an ML model that automates the approval decision while minimizing false approvals — making precision the key optimization metric.
🛠️ Tech Stack

Language: Python
Libraries: NumPy · Pandas · Scikit-learn · Matplotlib
Environment: JupyterLab

📁 Dataset

1,000+ real-world loan records
Format: CSV
Features include applicant income, credit history, loan amount, and more

🔄 ML Pipeline

Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Feature Engineering
Model Training & Evaluation
Performance Visualization

📈 Results
Naive Bayes outperformed other tested algorithms, achieving the best balance of accuracy and precision on this dataset.
